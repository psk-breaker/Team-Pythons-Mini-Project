## Terminal Café App
A Data Engineering Mini‑Project for Generations UK

## What is this?
The Terminal Café App is a command‑line ordering system built in Python. Developed as part of the Generations UK Data Engineering Bootcamp, it lets users browse a menu, build an order, calculate totals, and print a receipt—all directly in the terminal.

This project highlights fundamental programming concepts like data structures, control flow, functions, and error handling. It also features a modular design and uses a PostgreSQL database deployed on a RHEL virtual machine for data persistence.

## Objectives
* Create an interactive terminal application.
* Write clean, modular Python code.
* Manage menu items and prices using lists and dictionaries.
* Implement robust user input validation and error handling.
* Simulate a real‑world ordering process.
* Collaborate effectively through pair or group programming.
Key Features
* Menu Display: View available food and drinks alongside their prices.
* Ordering: Select multiple items and specify quantities.
* Basket Summary: Review your chosen items before paying.
* Totals Calculation: Automatically compute the final bill.
* Receipts: Generate an itemised receipt for your order.
* Session Control: Easily restart for a new order or exit the app.
